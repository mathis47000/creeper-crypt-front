creeper crypt front

# Docker:
- Build image:
```bash
docker build -t creeper-crypt-front .
```

- Run container:
```bash
docker run -p <port_you_want>:80 creeper-crypt-front
```

# Or use docker-compose:

- Run container (default port 1234):
```bash
docker-compose up
```
